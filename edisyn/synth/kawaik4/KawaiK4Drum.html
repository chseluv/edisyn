Kawai K4

Drum Patch Editor

This editor will work with the Kawai K4 and the Kawai K4r.

Communicating with Edisyn Set your K4 to receive Program Changes (RCV PGM) and Exclusive (RCV EXCL). See pages 79 and 80 of the manual. You'll also probably want your send channel and receive channel to be the same. I wouldn't set the receive channel to OMNI, as this might conflict with drums.

Hints

  • The Kawai K4/K4r receives drum information on a separate MIDI channel. You should change this MIDI channel in the patch editor so it's not the default MIDI channel you're using for your main sounds. Otherwise when you play a note via Edisyn it'll get played both as a drum and as your main sound, and you don't want that.

  • When you play a note (via the Send Test Note menu option) the note played is the currently selected drum key.

  • You can select a drum key either via the dial or by the on-screen keyboard. The widgets shown below reflect the settings for that particular key.

  • It's rare for you to want your drum keys to vary in their submix. So when you change the submix, you can press Apply to All to set all the drum keys to that new value.

  • The following effect patches are not used by any standard single, multi, or drum patch: 3, 10, 11, 12, 14, 15, 21, 22, 23, 28

  • In all standard effect patches, Submix H is always "off".

Gotchas

  • Unfortunately drums don't have their own effects/output: they will use whatever effect/output the current single or multi patch is using.

  • If you request a patch, the MIDI channel will not be properly updated. This is due to a Kawai K4 drum patch bug: it doesn't return the MIDI channel when it sends sysex data to Edisyn.

  • When you update a parameter, it's not reflected on the K4's LCD until you move to something else and back again.
