PreenFM2

Single Patch Editor

This editor will work with the PreenFM2. It probably won't work with the PreenFM.

Communicating with Edisyn Set up your PreenFM to emit NRPN (not CC) and to receive NRPN (or CC+NRPN). The PreenFM should also be set up to accept Program changes.

Gotchas The PreenFM2 has an unusual approach to upoading and downloading sounds which can be challenging to use with an editor. Specifically, it doesn't upload or download via sysex. Rather, Edisyn can send NRPN commands to it to change individual parameters, and there is a special NRPN command to cause the PreenFM2 to dump all its parameters to Edisyn via a deluge of NRPN commands. There is no way to write a patch: you have to save patches manually on the synthesizer. So:

  • When you request a specific Patch, Edisyn first changes to that patch, then loads it.

  • Similarly, when you send a specific Patch, Edisyn first changes to that patch, then sends it.

  • You cannot write patches. Instead, send the patch, then manually save it on the PreenFM2.
