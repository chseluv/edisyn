Waldorf Microwave II / XT / XTk

Multi Patch Editor

This editor will work with the Waldorf Microwave II, the Waldorf Microwave XT, and the Waldorf Microwave XTk in 10 voice or in 30 voice.

Communicating with Edisyn Set your Microwave to send Sysex only, not Ctl nor Ctl+Sysex. Also set Receive to on.

Show Each Instrument has a button called Show. This shows the given sound (bank and number) as a separate Blofeld Single editor panel.

XTk Options Two options are only available for the Microwave XTk:

  • MIDI Send (In General) Set this to Global to send MIDI on all instruments regardless of their individual settings. Set this to Specific to allow individual instruments to specify whether they want their MIDI sent.

  • MIDI Send (In each Instrument) determines whether a given instrument will send its MIDI out. This is overridden by the General option above.

Gotchas There are a few:

  • Edisyn cannot tell the Microwave to switch to a given multi patch number. As a result, *Send Patch* and *Send Patch To...* are disabled, a nd the editor is *never* in sync (because we don't know what patch number is on the machine). To get a patch, manually switch to the desired patch number, then call *Request Current Patch* (or dump the patch manually to Edisyn). To send a patch, set the patch location manually on your synthesizer, then use *Send Current Patch*.

  • Though the Microwave responds to individual parameter changes in Multimode, it does not send any parameter change sysex when you turn its knobs.

  • Also, when you make individual parameter changes, the Microwave's screen won't update to reflect those changes: but they've been made. Try scrolling away and then back, and you'll see the changes magically appear.

Fun Feature When your patch editor becomes the front window, it'll ask the Microwave to change to Multi mode automatically.
